Registration System - Luma Recommendation:

For registration, we recommend Luma (lu.ma) — it's purpose-built for events like this and free for free events.

Why Luma works:

  • Multiple ticket types (Firefly Residents, Influencers, General Public)
  • Built-in QR check-in app (iOS/Android)
  • One-time-use codes that can't be screenshot-shared
  • Collects name, email, phone
  • Free for free events

Setup steps:

  1. Create account at lu.ma
  2. Create the event (May 9, 2026, Eagle Mountain)
  3. Set up ticket types with different perks
  4. Add custom registration questions (phone number)
  5. Send us the embed code — we'll style it to match the site

Once you have Luma set up, we'll embed it directly into the site so registration feels seamless.

Let us know your thoughts on the design direction and any changes you'd like!
